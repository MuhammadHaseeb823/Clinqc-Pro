# ClinQC Pro

(https://doi.org/10.5281/zenodo.21348547)

## Overview

ClinQC Pro is a browser-based Clinical Biochemistry Internal Quality Control (IQC) training and research application.

## Features

- Westgard QC rule evaluation
- Levey-Jennings charts
- QC statistics
- Multi-analyte laboratory simulation
- Performance monitoring tools
- Progressive Web App (PWA) installation

## Important Notice

Synthetic / simulated data only — research & training tool, not validated for clinical decision-making.

## Live Application

https://muhammadhaseeb823.github.io/Clinqc-Pro/

## Citation

If you use ClinQC Pro in academic work, please cite the Zenodo DOI.
